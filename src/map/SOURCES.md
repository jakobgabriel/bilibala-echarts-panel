# Bundled map sources

This directory is regenerated by `scripts/fetch-maps.sh`. Each map's source
URL and license are listed below for attribution. Maps are fetched at
maintenance time and committed verbatim (minified) so the plugin has no
network dependency at build or runtime. Geometry is simplified by
[mapshaper](https://github.com/mbloch/mapshaper) to keep each file under
~600 KB; coordinate precision is reduced to 4 decimals (~11 m) which is
plenty for echarts at panel scale.

`china.json` predates this script — it was inherited from upstream
`bilibala-echarts-panel` and is preserved unchanged for backwards
compatibility with dashboards that bind data by Chinese-language province
names. `china-en.json` is the English-labeled sibling for international
dashboards (see [docs/maps.md](../../docs/maps.md#picking-the-right-china-map)
for a side-by-side comparison).

## Normalization

Every feature is post-processed by `fetch-maps.sh` to guarantee:

- `properties.name` — the English label (hoisted from source-specific
  keys like `NAME_EN`, `nom`, `reg_name`, `nam`).
- `properties.nameAscii` — a diacritic-stripped variant (`"Île-de-France"`
  → `"Ile-de-France"`) for variable-driven dashboards.

## Per-map notes

- **`usa.json`** — the upstream `click_that_hood/united-states.geojson`
  ships 49 features: 48 contiguous states + the District of Columbia.
  Alaska and Hawaii are absent and would need a custom GeoJSON drop-in.
- **`japan.json`** — administrative suffixes (`Ken`, `Fu`, `To`, `Do`)
  are stripped from prefecture names (`"Kyoto Fu"` → `"Kyoto"`).
- **`china-en.json`** — the upstream label `"Heilongjian"` is corrected
  to `"Heilongjiang"` during the fetch.
- **`world.json`** — Natural Earth 50m countries; coastline detail is
  preserved at the highest simplify percentage that fits the budget.
